# taskRecorder

## 应用说明

主要是用来记录一些 project 或者 task 的进度，并进行可视化

## 使用技术

+ flask 
+ flask-login 进行路由访问的限制
+ flask-wtf 处理表单
+ flask-sqlalchemy 处理数据库
+ pyecharts 可视化数据
